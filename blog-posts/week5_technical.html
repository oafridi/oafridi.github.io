Object Oriented programming in Ruby

Posted by Osman,

Let's dive straight into ruby classes. We're going to build a class that will help keep a record of students in our fictional high school - DBC High. The students will have various data associated with them, to keep it simple we're going to add just three to begin with. These will form the attributes of our class:

  • first name
  • last name
  • date of birth

Let's start by defining our class

             
              class Student                          
              end

Now when we create a new student we can instantiate this class. Instantiation is the way we create an instance of our class. We do this using the 'new' keyword. For example 'John Major' has just joined our school, let's create a new student: 

             
              john_major = Student.new

We now have a john_major object created from the Student class. However we don't have any of the attributes for John that we need - his date of birth, name etc. So how do we do that? Well these attributes will be passed to our class so we need some way of receiving them. In Ruby we can do this using an 'initialize' method which will take the values we pass our Class. Let's see how this works:

             
              class Student
                def initialize(first_name,last_name,dob)
                  @first_name = first_name
                  @last_name = last_name
                  @dob = dob
                end                          
              end

Notice the @ sign in front of some variables. These are instance variables and belong to the instance. Now let's go ahead and create the john_major object again but this time with the attributes: 

             
              john_major = Student.new("John", "Major", 35)

We now have a john_major object which contains data related to John. We can see his age easily using the following notation: john_major.dob => 01/18/1983
