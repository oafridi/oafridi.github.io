Overview

A brief comparison of the two types:

  • SQL databases are table based, NoSQL databases are document based
  • To query and interact with a SQL db we use SQL (structured query language), to interact with NoSQL we use database-specific syntax.
  • SQL dbs have predefined schemas NoSQL dbs do not require them.
  • NoSQL can provide benefits over SQL when used for big data in terms of speed and performance
