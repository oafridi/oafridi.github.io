Margin vs Border vs Padding

Posted by Osman, 2014-09-13

When it comes to CSS a conversation about margins, borders and padding would have to begin with a discussion about the box model. So what is it?

The box model is pretty simple as you can see from the image. And consists of three different parts:

  1. Margin - area around the border
  2. Border - goes around the padding and the content
  3. Padding - goes around the content
Take a look at the example CSS here: {% highlight css %} div { width: 220px; padding: 10px; border: 5px solid gray; margin: 0px; } {% endhighlight %} What's important to note is that despite the fact the width has been set to 220px the total width of the element should take into account the padding, the border and margin if they have been given values. You may be wondering why he have margins, borders and padding and when to use each. A quick rundown of the differences :
  • The border as its name would suggest can add a border around your element. This is useful if you want to box off certain elements or create a separation between elements.
  • The padding is the extra space between the border and the element. You might use this to create more of a space between the border and the element
  • The margin is the space outside of the border and any other elements. You might use this to create spacing between blog posts for example
