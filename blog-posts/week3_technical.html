Javascript

Posted by Osman, 23rd Aug 2014

Just why is JavaScript so popular in web development?

Javascript began as a client side language. Users could use scripts to interact with users control the browser and alter the document content. As its evolved the things you can do with Javascript has increased. There are numerous Javascript libraries such as Jquery which provide extra functionality to websites - a sliding carousel for example, or a way to respond to a user click by triggering an animation.

Beyond this Javascript is now also used as a server side language in Node.js and a lot of exciting stuff is happening in this space - single page websites for example

The good the bad and the ugly

There's a lot of good when it comes to Javascript including the functionality mentioned. Javascript as a language also supports multiple programming paradigms OOP, imperative, functional etc meaning that it can be suited to all types of inclinations and makes available a greater range of choices when considering implementation.

And so now the bad - Javascript has little type safety and code can easily have latent type errors which are not discovered until a site has been in production for several weeks. Security is also a huge concern with things like cross site scripting and forgery a constant concern that has to be kept in mind with coding.

Overall Javascript is ubiquitous and the web simply would not function nearly as well as it does without it (don't believe me - try disabling Javascript in your browser and see how you get by for the next week). Javascript is here to stay.
